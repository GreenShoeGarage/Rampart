Rampart FI 033
Physical Security Field Kit

A modular set of instruments for sanctioned physical security work. Every module is a self contained tool that shares one chassis. The kit assumes a scope, a signed authorization, and a finding worth writing down at the end.

Instruments

Operating note

Rampart is built for assessors working inside a written authorization. The kit is deliberately weighted toward survey, documentation, and reporting, which is where the value to a client actually lives. Findings point at the fix, not the break in.
