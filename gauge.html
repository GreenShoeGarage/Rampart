How to measure well

Place the reference object in the same plane as what you are measuring, for example flat against the same door or wall.
Shoot straight on. Perspective foreshortens anything tilted away from the lens and skews the result.
Set the reference across its longest clear edge for the best accuracy, then measure.
This is an estimate from a single photo, not survey grade. For anything load bearing on a decision, confirm with a tape.

Useful references

Credit or ID card, long edge 85.6 mm. US banknote, long edge 156 mm. Letter paper, long edge 279 mm. Standard brick, long edge 203 mm. Standard door height, 80 in. A tape measure or folding rule shot in plane is best of all.
